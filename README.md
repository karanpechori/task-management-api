# Todo API

Task Management Application using Node.js, showcasing expertise in RESTful API design, data modeling, and user authentication.

## Tech Stack Used

- Express
- MongoDB + Mongoose
- Zod
- TypeScript
- NodeJS

## Features

- RESTful API design pattern
- Error Handling
- Middlewares
- User Authentication

### Installation

Using pnpm

```sh
pnpm install
```

Using npm

```sh
npm install
```

### Setup

Example `.env` file

```.env
MONGO_DB_URI=mongodb://localhost
MONGO_DB_NAME=todo
```

### Running

using pnpm

```sh
pnpm start
```

using pnpm

```sh
npm run start
```
